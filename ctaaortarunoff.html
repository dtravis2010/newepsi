CTA RUNOFF WITH CONTRAST

 If an order says CTA AORTA RUNOFF then it is a valid order because the scan is done from the diaphragm all the way to the toes. But if the order says CTA AORTA and does not indicate a body part, then it is not a valid order – a body part must be indicted reason being that a CTA AORTA could be chest, abdomen & pelvis, abdomen & pelvis, or just abdomen. *Review huddle notes* 

 


A CTA is done WITH & WITHOUT contrast. If an order says WITH it is valid but transcribe WWO. If the order says WITHOUT, we need a revised order.
